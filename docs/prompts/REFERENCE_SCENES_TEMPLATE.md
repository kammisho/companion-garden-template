# REFERENCE SCENES TEMPLATE

新しい observation を、既存地層へ照らして読むための参照面です。
完全一致を探すのではなく、`どの層が立ったか` を見るために使います。

## 使い方

1. 新しい実例を、どの probe に近いかで見る
2. このファイルで対応する既存地層を開く
3. `同じ文` ではなく `同じ層` が立っているかを checklist で見る
4. 見えたことだけを observation log に短く残す
