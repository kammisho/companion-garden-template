# WORKLOG

このファイルは、再開できる粒度で作業を折り返すためのログです。

## 記録ひな形

- Date:
- Phase:
- What changed:
- Files:
- なぜ:
- Next:

## 最初のメモ

- Date:
  実際の作業が始まったら記入する
- Phase:
  骨組みづくり / 最初の companion の植え付け
- What changed:
  実際に作ったもの、並べ替えたものだけを書く
- Files:
  触ったファイルを書く
- なぜ:
  なぜこれが最小の一手だったかを書く
- Next:
  次に再開するときの具体的な一手を 1 つ残す
